Section

Extend from: https://bulma.io/documentation/layout/section

Basic Usage

.section

Modifier

.is-fit
.is-small
.is-medium
.is-large

Changes

Add is-fit, is-small

Custom section padding for mobile

Change section padding to have same left and right padding size for all modifier

Variables

$section-gap: 24px;

$section-gap-mobile: $section-gap;

$section-padding-fit: 0 $section-gap;

$section-padding-small: variables.$block-spacing $section-gap;

Bulma Variables

$section-padding-desktop: (variables.$block-spacing * 2) $section-gap;

$section-padding-medium: (variables.$block-spacing * 3) $section-gap;

$section-padding-large: (variables.$block-spacing * 6) $section-gap;
