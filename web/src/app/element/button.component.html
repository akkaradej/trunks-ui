Button

Extend from: https://bulma.io/documentation/elements/button

Basic Usage

Modifier

.is-[color]

.is-outlined

.is-loading

is-[size]

Changes

Set button font-size to $size-normal

Set button vertical padding to 0

Custom button horizontal padding

Custom button height

Add is-fat to has min width 100px

Add is-auto to has default padding and auto height

Variables

$button-font-size: variables.$size-normal;

$button-height: variables.$control-height;

$button-small-height: variables.$control-height - variables.$brick-spacing;

$button-small-padding-horizontal: 1rem;

$button-medium-height: variables.$control-height + variables.$brick-spacing;

$button-medium-padding-horizontal: 2rem;

$button-large-height: variables.$control-height + variables.$brick-spacing * 2;

$button-large-padding-horizontal: 3rem;

Bulma Variables

$button-weight: variables.$weight-normal;

$button-padding-horizontal: 1.5rem !default;
